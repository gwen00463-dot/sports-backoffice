# Sports Backoffice

스포츠 베팅 백오피스 대시보드 (단일 정적 HTML).

- **배포 파일**: `index.html` — Cloudflare Pages가 이 파일을 도메인 루트로 서빙합니다.
- 작업 원본은 로컬의 `sports-backoffice (1).html`이며, 배포 시 `index.html`로 복사됩니다.

## 업데이트 방법
1. 원본(`sports-backoffice (1).html`) 수정
2. `index.html`로 복사
3. `git add index.html && git commit -m "update" && git push`
4. Cloudflare Pages가 자동으로 재배포 → 도메인에 반영
